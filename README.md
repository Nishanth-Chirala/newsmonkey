# 📰 NewsMonkey

A modern React-based news application that delivers real-time headlines using NewsAPI.

---

## 🚀 Features

* 🔹 Real-time news fetching using NewsAPI
* 🔹 Category-based browsing (Business, Entertainment, Health, Science, Sports, Technology)
* 🔹 Infinite scrolling for seamless experience
* 🔹 Fully responsive design
* 🔹 Reusable React components

---

## 🛠️ Tech Stack

* React.js
* JavaScript
* HTML & CSS
* Bootstrap
* NewsAPI

```bash
cd newsmonkey
npm install
npm start
```
